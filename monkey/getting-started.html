Device updates

It is recommended to update your device to the newest firmare version. Updates are easily handled in the browser, on the device update page. If there is a new update available, update it with a single press of a button.

Connecting up

Monkey supports a variety of hosts:

Windows currently does not have the best support for MIDI over Bluetooth. For now, this is the recommended way to connect MONKEY to Windows:
  1. Connect Monkey to Windows through Bluetooth settings: If you are using Windows 11, you need to first enable advanced Bluetooth scanning in Bluetooth settings. Just after its paired it will automatically disconnect, and is expected behaviour. I interpret this as: windows recognises that no programs uses the bluetooth device and thus just disconnects from it to save power
  2. Create a virtual MIDI port:Now we need to create a virtual MIDI port which we can connect together with out physical MIDI device. We might do this using loopMIDI, which can be downloaded from here: https://www.tobias-erichsen.de/software/loopmidi.html
  3. Connect device with virtual MIDI port: Install MIDIBerry and select WAVY MONKEY as input device and the virtual MIDI port you created as the output port. Now Windows should again reconnect to MONKEY and your ready to use it in your favorite DAW! Download MIDIBerry here: https://www.microsoft.com/store/apps/9n39720h2m05

Configure DAW

By default Monkey sends on MIDI channel 1 for instruments and 16 for the drums. The instrument channel is changeable by pressing SHIFT+CH, then while keeping SHIFT pressed, press one of the numbered buttons.

Make sure Monkey is enabled in settings
